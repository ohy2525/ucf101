# ucf101
# 데이터셋
- 101 카테고리의 동작을 행동 인식을 위해 유튜브에서 실제 움직을 녹화한 데이터  
- UCF 101은 다양성 동작을 제공하며 카메라 모션, 물체 모양 및 포즈, 물체 크기, 시점, 배경, 조명 등 현재까지 다양한 데이터 셋을 갖추고 있음
